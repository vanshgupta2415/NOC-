# Online No Dues Portal - MITS Gwalior

A secure, role-based backend and frontend system for automating the Online No Dues Portal for MITS Gwalior.

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, TanStack Query, Framer Motion
- **Backend**: Node.js, Express, MongoDB
- **Automation**: Email delivery (Nodemailer), PDF Generation

## Project Structure
- `nodues backend/`: Express server and MongoDB models
- `Nodues frontend/`: React application

## Getting Started
1. Install dependencies in both folders: `npm install`
2. Set up environment variables in `.env`
3. Run dev servers: `npm run dev`
